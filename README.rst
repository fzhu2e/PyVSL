**********
PyVSL
**********

VS-Lite in Python. It contains two modes:

1. A python wrapper of `the R package of VS-Lite <https://github.com/suztolwinskiward/VSLiteR>`_ by Suz Tolwinski-Ward.
2. VS-Lite in pure Python (TODO).
