PyVSL

VS-Lite in Python. It contains two modes:

  1. The wrapper mode. PyVSL implements a python wrapper for VS-Lite in R and another for VS-Lite in Matlab , along with a wrapper for the Bayesian estimator for the parameters (in Matlab)

  2. VS-Lite in pure Python (TODO; welcome to contribute).
